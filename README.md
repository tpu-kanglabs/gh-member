# gh-member

GitHub CLI extension for managing organization members.

## Requirements

- [GitHub CLI](https://cli.github.com/)
- [mise](https://mise.jdx.dev/) (Only for development)

## Installation

```bash
gh extension install tpu-kanglabs/gh-member
```

## Usage

```bash
gh member list [org]
```

For full command documentation — tutorials, how-to guides, flag reference, and explanation of output modes — see **[docs/commands.md](docs/commands.md)**.

### Upgrade

```bash
gh extension upgrade member
```

### Uninstall

```bash
gh extension remove member
```
