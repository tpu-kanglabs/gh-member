# gh-member

GitHub CLI extension for managing organization members.

## Requirements

- [GitHub CLI](https://cli.github.com/)
- [mise](https://mise.jdx.dev/) (Only for development)

## Installation

```bash
gh extension install tpu-kanglabs/gh-member
```

## Usage

```bash
gh member
```

### Upgrade

```bash
gh extension upgrade member
```

### Uninstall

```bash
gh extension remove member
```
